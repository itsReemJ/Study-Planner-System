# Study Planner System

A PHP and MySQL web application that helps students manage subjects, tasks, deadlines, and profile information.

## Features

- User registration, login, and logout
- Dashboard with study statistics
- Add and delete subjects
- Add, complete, and delete tasks
- Update profile information
- Upload a profile picture
- AJAX-based subject and task updates

## Technologies

- PHP
- MySQL
- HTML and CSS
- JavaScript and AJAX
- XAMPP and phpMyAdmin

## How to Run

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project folder into the XAMPP `htdocs` folder.
4. Open phpMyAdmin.
5. Create a database named `study_planner`.
6. Import `database.sql` into the database.
7. Open the project in the browser:

   `http://localhost/STUDY-PLANNER-SYSTEM/`

The default database settings in `config.php` are:

- Host: `localhost`
- Username: `root`
- Password: empty
- Database: `study_planner`
